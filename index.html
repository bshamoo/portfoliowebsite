    PROJECT01 - Portfolio Website

    One of my goals for Summer 2020 was to develop and establish my personal brand. To achieve this, I decided to create a portfolio website where I can showcase myself through my passions, work, skills, and accomplishments. This website stands as a project reflecting my curiosity to learn new skills and determination for self-improvement.

    To efficiently develop this project from start to finish and because I had a strong understanding of the requirements, I decided to follow the waterfall model as a general guideline for my development process. For the design phase, I used Figma, an interface design tool, to visualize all my requirements and create wireframes for each webpage. (SLIDES 01 and 02) To implement these designs into a fully functioning website, I first had to teach myself how to code in HTML and SCSS. After many hours of watching and reading tutorials on website coding basics, I decided to implement Bootstrap to help with the responsitivity of my website. During my development process, I used a live server/real-time SASS compiler through Visual Studio Code to test and debug along with a GitHub repository to keep track of each iteration.

    Tools Used: HTML , SCSS , Bootstrap , JavaScript , Figma , Visual Studio Code, GitHub
